# Milestone 01 — First Playable Prototype

Working title: **Lumi's Leap** (a fully original character — feel free to rename).

## What we built
- A scalable project folder structure.
- Placeholder art (player, ground, platform, coin, sky) generated by a Python script.
- A reusable **Player** scene with running, gravity, single + double jump, and a follow camera.
- A **Level_01** scene with a ground floor and three floating platforms to test jumping.

## Controls
- Move: **A / D** or **Left / Right arrows**
- Jump / Double jump: **Space** or **Up arrow**

## Folder structure
```
assets/        placeholder art (PNG)
audio/         sound effects + music (later)
characters/    Player.tscn + Player.gd
enemies/       enemy scenes (later)
levels/        Level_01.tscn (the playable scene)
ui/            menus + HUD (later)
scripts/       shared/utility scripts (incl. the asset generator)
effects/       particles (later)
animations/    animation resources (later)
save_system/   save/load (later)
managers/      game/audio/scene managers (later)
docs/          notes like this file
```

## Key concepts you learned
- **Scene**: a saved tree of nodes (.tscn). Player and Level are both scenes.
- **Node**: a single building block (a sprite, a body, a camera...).
- **CharacterBody2D**: a body built for player-controlled movement.
- **StaticBody2D**: a body that never moves (ground/platforms).
- **CollisionShape2D**: the invisible shape used for physics collisions.
- **_physics_process(delta)**: runs at a fixed rate; where movement/physics belong.
- **velocity + move_and_slide()**: you set a speed, Godot moves the body and handles collisions.
- **Coyote time / jump buffering**: tiny forgiveness windows that make jumping feel good.

## How to run
Open the project in Godot and press **F5** (Run Project). Level_01 is the main scene.
